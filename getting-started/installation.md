# Installation

No installation is required for GitBook's web editor. However, if you want 
to edit locally:

## Local Editing Setup

1. Clone your repository:
```bash
   git clone https://github.com/YOUR_USERNAME/gitbook-docs.git
```

2. Edit any `.md` file using VS Code or any text editor

3. Push changes back to GitHub:
```bash
   git add .
   git commit -m "Update docs"
   git push
```

GitBook will automatically detect the changes and update your published site.
