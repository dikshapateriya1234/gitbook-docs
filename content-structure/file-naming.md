# File Naming Conventions

## Rules to Follow

- Use lowercase only: `quick-start.md` not `QuickStart.md`
- Use hyphens for spaces: `file-naming.md` not `file_naming.md`
- Be descriptive: `github-sync.md` not `sync.md`
- Always use the `.md` extension

## Examples

| Good ✅ | Bad ❌ |
|--------|-------|
| `quick-start.md` | `QuickStart.md` |
| `folder-organization.md` | `folder_org.md` |
| `github-sync.md` | `sync.md` |
| `publishing.md` | `pub.md` |
