# 📚 GitBook Documentation Guide

Welcome to this documentation site! This guide explains how to structure 
content in GitHub so GitBook can read it, and how to create clear navigation.

## What You'll Learn

- How to connect GitBook to GitHub
- How to structure your files and folders
- How to create left sidebar and right-side navigation
- How to publish your documentation publicly

## Why GitBook + GitHub?

GitBook reads your Markdown files directly from your GitHub repository. 
Every time you push changes to GitHub, GitBook automatically updates your 
published documentation.
