# Left Sidebar Navigation

The left sidebar is controlled entirely by SUMMARY.md.

## How to Create Sections

Use double hash marks to create section headings in SUMMARY.md. 
These appear as bold labels in the sidebar and are not clickable.

## How to Create Links

Use asterisk followed by brackets and parentheses to create page links. 
Indent with two spaces to create nested links under a parent page.

## Rules to Follow

1. Section headings are created with double hash in SUMMARY.md
2. All links use relative file paths
3. Nested items are indented with two spaces
4. Every file you want visible must be listed here

## Result in Sidebar

Your sidebar will show sections as bold labels with clickable 
page links underneath them, and nested pages indented below their parents.
